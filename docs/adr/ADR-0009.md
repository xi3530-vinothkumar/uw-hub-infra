# ADR-0009: Postgres system-of-record owned by Java only

**Status:** Accepted

## Context
State must be durable and the audit trail permanent. Python must stay stateless (ADR-0002).

## Decision
PostgreSQL is the single source of truth, accessed only by Java. It stores submissions,
COPE, photos/findings, peril exposures, versioned decisions, audit entries, task-result
idempotency records, and append-only event logs. Photo binaries live in disk/MinIO with a
reference in Postgres.

## Consequences
+ Clear ownership; resumable/idempotent pipeline; durable audit.
- Data-at-rest encryption + retention deferred to production (business-confidential data).

## Alternatives considered
- Shared DB access from Python: rejected — breaks the stateless-worker boundary.
