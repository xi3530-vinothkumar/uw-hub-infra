# ADR-0005: Hybrid real + mocked peril data sources

**Status:** Accepted

## Context
Real free public hazard data exists but varies in usability. Some are clean point-APIs;
others are bulk datasets requiring heavy pre-processing.

## Decision
Use real free no-key APIs where feasible — Nominatim (geocoding), FEMA NFHL (flood), USGS
Seismic Design Maps (earthquake). Keep hurricane/wind/hail (NOAA bulk CSV) and wildfire
(no free point-API) mocked, behind the same `PerilSource` interface with a `source` label.

## Consequences
+ Credible demo with real federal data; honest, evaluated tradeoff per source.
+ Swappable interface — mocks become real later with no caller change.
- Two integration styles to maintain; provenance must be surfaced in the UI.

## Alternatives considered
- All mocked: weaker credibility. All real: NOAA/wildfire not feasible in the timeline.
