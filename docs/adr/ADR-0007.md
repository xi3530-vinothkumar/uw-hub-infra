# ADR-0007: Deterministic additive scoring model + gates

**Status:** Accepted

## Context
Per ADR-0001 the score must be deterministic, explainable, and tunable.

## Decision
An additive model: baseline 20 + signed bounded contributions (construction class/age/roof,
occupancy tier, sprinkler/alarm credits, governing + secondary peril, photo condition),
clamped 0–100. Bands: Accept 0–39, Refer 40–69, Decline 70–100. Gates (severe peril, low
confidence on a critical field, missing TIV) adjust upward only / fail explicitly. Weights
and thresholds are centralized constants. See HLD §12.

## Consequences
+ Every factor emits one audit entry; fully reconstructable; unit-testable.
+ Tunable against loss data without structural change.
- Weights are an initial calibration, not empirically fitted (noted as production work).

## Alternatives considered
- ML risk model: out of scope, not explainable enough for the MVP, no training data.
